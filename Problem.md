| 问题         | 定义                                                        | 应用                                 |
| ------------ | ----------------------------------------------------------- | ------------------------------------ |
| 分类         | $x \rightarrow c$                                           | 文本分类、情感分析                   |
| 匹配         | $x,y \rightarrow R$                                         | 搜索、问答、单轮对话（基于检索）     |
| 转换         | $x \rightarrow y$                                           | 机器翻译、摘要、单论对话（基于生成） |
| 结构预测     | $x \rightarrow [x]$                                         | 序列标注、中文分词、语义分析         |
| 序列决策过程 | $\pi:s \rightarrow a$, 在多个字符串表示的状态$s$选择动作$a$ | 任务驱动多轮对话                     |
